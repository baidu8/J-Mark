📦 J-Mark
一款由“江大爷”开发的极简、纯净、高颜值的 Chrome 书签导出工具。

✨ 核心功能
一键导出：精准选择文件夹，导出为标准 data.js 格式。

搜索过滤：实时搜索，秒找目标。

极简 UI：现代大圆角设计，无滚动条干扰。

安全隐私：100% 本地运行，不传输任何数据。

🚀 快速开始
下载本项目并解压。

打开 Chrome chrome://extensions/，开启开发者模式。

点击 加载解压的扩展程序，选择 J-Mark 根目录。

固定插件，即刻开始高效管理书签。

📖 使用指南
点击插件图标弹出界面。

使用搜索框定位特定文件夹，或直接在列表勾选/取消勾选。

点击 “导出并下载数据” 按钮。

将生成的 data.js 放到你导航页项目的指定目录下即可完成更新。

🛠️ 技术实现
Manifest V3: 采用最新的 Chrome 扩展标准。

深度优先搜索 (DFS)：递归算法确保书签树级联逻辑准确。

原生 JS: 无第三方库依赖，极轻量，加载速度秒开。
